Запуск проекта посредством gulp из консоли
npm i - установка зависимостей.
npm run start - старт сервера.
npm run build - сборка проекта для его последующего запуска(Только сборка, т.е. будут созданы необходимые файлы но сервер запущен не будет).

Данное задание выполенно без шапки и подвала.
Применены:
npm + gulp + webpack
Использовался flex, грид-сетка ввиду объективных причин не применялась.
За пример стилистики принят сайт www.quarta.su
Форма сделана адаптивной от разрешения 320px.
Инлайновые изображения не использовались, соответственно и спрайт не создавался.
Показано применение фоном изображения для кастомного чекбокса.
Учтена валидация поля телефона и добавлена невозможность изменения полей (визуальное определяемое) и отправки формы в случае не согласия
 с политикой портала.
Документ проходит провекри на валидность кода (с заложенным принятым исключением для
 общего класса visually hidden).
